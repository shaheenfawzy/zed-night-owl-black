# Night Owl Black for Zed

A pure black variant of the Night Owl theme, ported to [Zed](https://zed.dev).

## Description

Night Owl Black is a dark theme optimized for OLED displays and fans of pure black backgrounds. It features the same beautiful syntax colors as Night Owl but with a true black (`#000000`) editor background and dark grey UI elements for increased contrast.

## Installation

1. Open Zed
2. Open the Extensions panel (`cmd+shift+x` on macOS, `ctrl+shift+x` on Linux)
3. Search for "Night Owl Black"
4. Click Install

## Color Palette

| Element | Color |
